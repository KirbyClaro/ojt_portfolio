About This Portfolio

This web application serves as both my professional resume and a functional document management system for my On-the-Job Training (OJT). Rather than utilizing pre-made templates, I developed this platform entirely from scratch to demonstrate practical competency in front-end web development and cloud backend integration.

  • Frontend Architecture: Developed using standard HTML5, CSS3, and vanilla JavaScript for a lightweight, responsive, and tailored user interface.
  • Cloud Backend (Google Firebase): Fully integrated with Firebase as a serverless backend to handle data processing and user actions securely.
  • Real-Time Database Management: Utilizes Firebase Firestore (a NoSQL document database) for real-time tracking, rendering, and timestamping of weekly progress logs and document metadata.
  • Secure Cloud Storage: Leverages Firebase Storage to securely host and dynamically retrieve uploaded OJT requirements (such as PDFs) and weekly report photo galleries.
  • Core Capabilities: Features password-protected file uploading, secure document and log deletion controls, dynamic interactive galleries with hover-zoom capabilities, and real-time database syncing without requiring page reloads.

National Telecommunications Commission

National Capital Region (NCR) Branch

The NTC is the primary government agency entrusted with the oversight, adjudication, and control of all telecommunications services, television, and radio networks throughout the Philippines.

The IT Unit’s main daily function is to streamline institutional operations by maintaining critical hardware, managing database systems for licensing and requests, and developing custom software solutions to automate administrative workflows. Agency Mandate & Core Values

Core Functions:

  • Regulate the installation, operation, and maintenance of radio stations for both private and public use.
  • Regulate and supervise the provision of public telecommunications services.
  • Manage the national radio frequency spectrum.
  • Regulate and supervise radio and television broadcast stations, including cable (CATV) and pay television.

Core Values:

  • Regulation and Supervision: Ensuring that telecommunications services operate efficiently and fairly, adhering to national laws and policies.
  • Consumer Protection: Protecting consumers through fair competition and compliance with service standards.
  • Quality Service: Promoting quality service and ensuring that all telecommunications providers meet established standards.
  • Radio Spectrum Management: Efficiently managing the radio frequency spectrum to prevent interference and optimize bandwidth.
  • Adaptability: Adapting to new challenges in privacy, cybersecurity, and connectivity to ensure a secure and reliable telecommunications environment.
